# Spring Cloud Gateway

## 실행 방법

0. 각각의 서버를 실행한다.
1. 게이트웨이 역할 서버에 설정된 `path`가 포함한 요청을 전송한다.
2. 설정에 따라, 연결된 서버로 해당 요청이 라우팅되고, 그 결과를 응답으로 받을 수 있다.
